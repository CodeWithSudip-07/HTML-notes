🌐 HTML : Hyper Text Markup Language

  • Hypertext :- Text that contains links (called hyperlinks) to other text, documents, or resources.
  • Markup Language :- A computer language that uses tags to define the structure and presentation of text/documents (e.g., HTML).

Together, HTML = Hypertext Markup Language.

The world First website: https://info.cern.ch/hypertext/WWW/TheProject.html

1. The Heading (<h1> to <h6>)

👉 What it is: Used to display headings in HTML

  • <h1>: The most important heading, typically used only once per page for the main title.
  • <h2>: A major section heading.
  • <h3>: A sub-section heading under an <h2>.
  • ...and so on, down to <h6>, the least important heading.

2. The Paragraph (<p>)

👉 What it is: The <p> tag is used for writing paragraph. It's for grouping sentences and blocks of text together.

3. HTML Horizontal Rules(<hr>)

👉 What it is: The <hr> tag is used to insert a horizontal line (thematic break) on a webpage. It is mainly used to separate content sections.

4. The Line Break Tag: (<br>)

👉 What it is: The <br> tag is used to insert a line break inside text or content. It does not create extra space like <p>, just moves the text to the next line.

5. The Anchor Tag (<a>)

👉 What it is: The <a> tag is used to create a hyperlink that connects one page to another, or links to a section, email, or file.

Target attribute in <a> tag (short note):

The target attribute defines where the linked page will open.

  • _self : Opens the link in the same tab/window (default).
  • _blank : Opens the link in a new tab/window.

6. The Image Tag (<img>)

👉 What it is: The <img> tag is used to display images on a webpage. It is an empty tag (no closing tag) and requires the src attribute for the image path and alt for alternative text.
