🌐 HTML MEDIA

In HTML, media refers to elements that let you add and control multimedia content like audio, video, and images on a webpage. These features make websites more interactive and engaging.


📗HTML <video> Element:-

  • The <video> element is used to embed video files directly into web pages, offering built-in support for playback control, captions, autoplay, and styling.
  • Supported video formats usually include MP4, WebM, and OGG.

  • Attributes:

    1. controls: Displays play/pause and other media controls.
    2. autoplay: Starts playback when the page loads.
    3. loop: Repeats the video automatically.
    4. muted: Mutes the audio track by default.
    5. poster: Specifies an image to display before playback starts.

📗HTML <iframe> Element:-

  • The <iframe> tag allows embedding another HTML document (such as a video, map, form, or another web page) within the current page.

  • Common uses include:
    1. Embedding YouTube videos
    2. Integrating live maps or ads
    3. Displaying dynamic or external web content

    1. src: Source URL of the embedded content.
    2. width, height: Sets dimensions.
    3. title: Accessibility description.
    4. allowfullscreen: Enables full-screen mode for video.
    5. sandbox: Restricts embedded content for security.
    6. Other less common attributes include name, loading, and allow for permissions.
